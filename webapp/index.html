Python Distributed Evolutionary optimization on Google App Engine

  • GET
      • /get/pop/ -> return current population
      /get/job/ -> return a new job (not running)
      /get/jobs/ -> return all jobs
  • PUT
      • /put/job/ -> put a single job (saves IP from http request), use from VM
      /put/jobs/ -> update all Job's, use from driver
      /put/pop/ -> update the current population
