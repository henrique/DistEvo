Python Distributed Evolutionary optimization on Google App Engine

  • GET
      • /get/vm/ -> return a single VM, only if request.ip = (vm.ip from db)
      /get/vms/ -> return all active VM's
      /get/job/ -> return a new job (not running)
      /get/jobs/ -> return all jobs
  • PUT
      • /put/vm/ -> put a single VM (saves IP from http request), use from VM
      /put/job/ -> put a single job (saves IP from http request), use from VM
      /put/vms/ -> put all VM's (saves IP from json), use from driver
      /put/jobs/ -> put all VM's (saves IP from json), use from driver
